Interoperable

Built around a simple DSL, easily integrate your own forensic data collection tools. Forensibus supports right out of the box Velociraptor, DFIR-ORC, UAC and KAPE.

Blazingly fast ⚡️

Parses data in seconds, not hours! If load suddenly increases, forensibus handles it linearly.

Modular

Forensibus makes it easy to add new parsers or parsing plug-ins.

Scalability

Forensibus is easy to deploy at any scale. Y can start with a standalone binary and then end up with a distributed installation on a cluster.

Simple

Keep it simple, stupid! Analysts should be able to find what they want from the system without understanding internal system architecture nor reading the code.
